## About
Advanced weather trend forecasting using machine learning
on the Global Weather Repository dataset (127,451 readings).

## How to Run
1. Open the .ipynb file in Google Colab
2. Upload GlobalWeatherRepository.csv from Kaggle
3. Run all cells top to bottom

## Dataset
https://www.kaggle.com/datasets/nelgiriyewithana/global-weather-repository


## Models Built
| Model | R² Score |
|---|---|
| Random Forest | 0.86 |
| XGBoost | 0.91  Best |
| Ensemble | 0.89 |

## Key Findings
- Latitude is the #1 predictor of temperature
- UV index is the #2 predictor
- Anomalies cluster in peak summer and winter months
- Middle East countries have highest average temperatures
